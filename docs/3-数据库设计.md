# 数据库设计文档 (PostgreSQL)

## 1. 数据库概述

- **数据库类型**: PostgreSQL 15
- **字符集**: UTF8MB4 / UTF-8
- **时区**: Asia/Shanghai (GMT+8)

## 3. 表结构

### 3.1 project - 项目表

```sql
CREATE TABLE project (
    id BIGSERIAL PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    cover VARCHAR(500),
    style_id BIGINT,
    status SMALLINT DEFAULT 0,  -- 0-草稿 1-处理中 2-已完成
    consistency INTEGER DEFAULT 0,
    chapter_count INTEGER DEFAULT 0,
    progress INTEGER DEFAULT 0,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_project_status ON project(status);
```

### 3.2 episode - 章节表

```sql
CREATE TABLE episode (
    id BIGSERIAL PRIMARY KEY,
    project_id BIGINT NOT NULL REFERENCES project(id) ON DELETE CASCADE,
    chapter_number INTEGER NOT NULL,
    name VARCHAR(100) NOT NULL,
    content TEXT,
    status SMALLINT DEFAULT 0,  -- 0-待开始 1-处理中 2-已完成
    progress INTEGER DEFAULT 0,
    current_step SMALLINT DEFAULT 0,
    -- current_step: 0-输入内容 1-提取信息 2-生成图片
    --               4-固定角色 5-生成分镜 6-生成视频 7-合成完成
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_episode_project_id ON episode(project_id);
CREATE INDEX idx_episode_chapter_number ON episode(chapter_number);
```

### 3.3 character - 角色表

```sql
CREATE TABLE character (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL REFERENCES episode(id) ON DELETE CASCADE,
    project_id BIGINT NOT NULL REFERENCES project(id) ON DELETE CASCADE,
    name VARCHAR(50) NOT NULL,
    description VARCHAR(500),
    avatar VARCHAR(1024),
    video_url VARCHAR(500),
    features JSONB,  -- 角色特征描述(用于一致性生成)
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_character_episode_id ON character(episode_id);
CREATE INDEX idx_character_project_id ON character(project_id);
```

### 3.4 scene - 场景表

```sql
CREATE TABLE scene (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL REFERENCES episode(id) ON DELETE CASCADE,
    project_id BIGINT NOT NULL REFERENCES project(id) ON DELETE CASCADE,
    name VARCHAR(100) NOT NULL,
    description TEXT,
    thumbnail VARCHAR(500),
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_scene_project_id ON scene(project_id);
CREATE INDEX idx_scene_episode_id ON scene(episode_id);
```

### 3.5 storyboard - 分镜表

```sql
CREATE TABLE storyboard (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL REFERENCES episode(id) ON DELETE CASCADE,
    shot_number INTEGER NOT NULL,
    shot_type SMALLINT DEFAULT 1,  -- 1-远景 2-中景 3-近景 4-特写
    duration INTEGER DEFAULT 5,    -- 秒
    description TEXT NOT NULL,
    dialogue TEXT,
    thumbnail VARCHAR(500),
    video_url VARCHAR(500),
    image_prompt TEXT,
    video_prompt TEXT,
    order_index INTEGER DEFAULT 0,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_storyboard_episode_id ON storyboard(episode_id);
CREATE INDEX idx_storyboard_shot_number ON storyboard(shot_number);
```

### 3.6 storyboard_character - 分镜角色关联表

```sql
CREATE TABLE storyboard_character (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL,
    storyboard_id BIGINT NOT NULL,
    character_id BIGINT NOT NULL,
    project_id BIGINT NOT NULL,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_sc_storyboard_id ON storyboard_character(storyboard_id);
CREATE INDEX idx_sc_character_id ON storyboard_character(character_id);
```

### 3.7 storyboard_scene - 分镜场景关联表

```sql
CREATE TABLE storyboard_scene (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL,
    storyboard_id BIGINT NOT NULL,
    scene_id BIGINT NOT NULL,
    project_id BIGINT NOT NULL,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_ss_storyboard_id ON storyboard_scene(storyboard_id);
CREATE INDEX idx_ss_scene_id ON storyboard_scene(scene_id);
```

### 3.8 generated_image - 生成图片表

```sql
CREATE TABLE generated_image (
    id BIGSERIAL PRIMARY KEY,
    episode_id BIGINT NOT NULL REFERENCES episode(id) ON DELETE CASCADE,
    image_type SMALLINT DEFAULT 1,  -- 1-角色 2-场景
    target_id BIGINT,                 -- 关联角色或场景的ID
    image_url VARCHAR(500) NOT NULL,
    prompt TEXT,
    status SMALLINT DEFAULT 1,  -- 0-生成中 1-成功 2-失败
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_gi_episode_id ON generated_image(episode_id);
CREATE INDEX idx_gi_image_type ON generated_image(image_type);
```

### 3.9 video_task - 视频任务表

```sql
CREATE TABLE video_task (
    id BIGSERIAL PRIMARY KEY,
    project_id BIGINT NOT NULL,
    instance_id BIGINT NOT NULL,       -- 模型实例ID
    target_id BIGINT NOT NULL,         -- 角色ID或分镜ID
    task_id VARCHAR(50) DEFAULT '',
    task_type SMALLINT DEFAULT 1,      -- 1-角色视频 2-分镜视频 3-音视频合成
    status VARCHAR(10),
    progress INTEGER DEFAULT 0,
    overall_progress INTEGER DEFAULT 0,
    video_url VARCHAR(500),
    error_message TEXT,
    next_poll_at TIMESTAMP,
    poll_count INTEGER,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_vt_project_id ON video_task(project_id);
CREATE INDEX idx_vt_task_id ON video_task(task_id);
```

### 3.10 model_definition - 模型定义表

```sql
CREATE TABLE model_definition (
    id BIGSERIAL PRIMARY KEY,
    provider_code VARCHAR(50) NOT NULL,  -- openai / aliyun / baidu / volcengine
    base_url VARCHAR(255),
    status SMALLINT DEFAULT 1,  -- 1启用 0禁用
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 3.11 model_instance - 模型实例表

```sql
CREATE TABLE model_instance (
    id BIGSERIAL PRIMARY KEY,
    model_def_id BIGINT NOT NULL REFERENCES model_definition(id),
    model_code VARCHAR(100) NOT NULL,     -- gpt-4o / qwen-max
    model_type SMALLINT NOT NULL DEFAULT 1, -- 1-TEXT 2-IMAGE 3-VIDEO 4-AUDIO
    instance_name VARCHAR(100) NOT NULL,
    scene_code SMALLINT NOT NULL,          -- 1-通用文本 2-信息提取 3-角色生成...
    path VARCHAR(50),
    api_key VARCHAR(255) NOT NULL,
    params JSONB,
    status SMALLINT DEFAULT 1,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_mi_model_type ON model_instance(model_type);
CREATE INDEX idx_mi_scene_code ON model_instance(scene_code);
```

### 3.12 model_instance_default - 默认模型配置表

```sql
CREATE TABLE model_instance_default (
    id BIGSERIAL PRIMARY KEY,
    model_type SMALLINT NOT NULL,         -- 1-TEXT 2-IMAGE 3-VIDEO 4-AUDIO
    model_instance_id BIGINT NOT NULL,
    status SMALLINT DEFAULT 1,
    remark VARCHAR(255),
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE UNIQUE INDEX idx_mid_model_type ON model_instance_default(model_type) WHERE is_deleted = 0;
```

### 3.13 prompt_template - 提示词模板表

```sql
CREATE TABLE prompt_template (
    id BIGSERIAL PRIMARY KEY,
    prompt_code VARCHAR(64) NOT NULL UNIQUE,
    prompt_name VARCHAR(128) NOT NULL,
    scene_code SMALLINT NOT NULL,   -- SceneCode枚举
    prompt_content TEXT NOT NULL,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE UNIQUE INDEX idx_pt_prompt_code ON prompt_template(prompt_code);
```

### 3.14 prompt_template_default - 默认提示词配置表

```sql
CREATE TABLE prompt_template_default (
    id BIGSERIAL PRIMARY KEY,
    prompt_template_id BIGINT NOT NULL,
    status SMALLINT DEFAULT 1,
    scene_code SMALLINT NOT NULL,
    remark VARCHAR(255),
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE UNIQUE INDEX idx_ptd_scene_code ON prompt_template_default(scene_code) WHERE is_deleted = 0;
```

### 3.15 style_template - 风格模板表

```sql
CREATE TABLE style_template (
    id BIGSERIAL PRIMARY KEY,
    name VARCHAR(50) NOT NULL,
    description VARCHAR(500) NOT NULL,
    preview VARCHAR(500),
    parameters JSONB,
    is_default SMALLINT DEFAULT 0,
    is_deleted SMALLINT DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 4. 枚举值说明

### 4.1 项目状态 (project.status)

| 值 | 描述 |
|----|------|
| 0 | 草稿 |
| 1 | 处理中 |
| 2 | 已完成 |

### 4.2 章节步骤 (episode.current_step)

| 值 | 描述 |
|----|------|
| 0 | 输入内容 |
| 1 | 提取信息 |
| 2 | 生成图片 |
| 4 | 固定角色 |
| 5 | 生成分镜 |
| 6 | 生成视频 |
| 7 | 合成完成 |

### 4.3 模型类型 (model_instance.model_type)

| 值 | 描述 |
|----|------|
| 1 | TEXT |
| 2 | IMAGE |
| 3 | VIDEO |
| 4 | AUDIO |

### 4.4 场景代码 (scene_code)

| 值 | 描述 |
|----|------|
| 1 | 通用文本生成 |
| 2 | 信息提取 |
| 3 | 角色图片生成 |
| 4 | 分镜生成 |
| 5 | 分镜视频生成 |
| 6 | 语音合成 |
| 7 | 首帧图片生成 |

## 5. 软删除

所有业务表都包含:
- `is_deleted SMALLINT DEFAULT 0`
- `WHERE is_deleted = 0` 软删除过滤

## 6. 时间戳

所有表包含:
- `created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP`
- `updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP`
